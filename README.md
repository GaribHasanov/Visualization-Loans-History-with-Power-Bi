# Visualization – Loans History with Power BI

## Dashboards and Reports

This project presents an interactive **Power BI dashboard solution** for analyzing historical loan data, including disbursed loans and received repayments.

The objective is to provide clear financial insights through data visualization, enabling better understanding of loan performance and repayment behavior.

---

## Project Overview

This dashboard project focuses on two key financial perspectives:

- Loan disbursement analysis  
- Loan repayment tracking  

It helps stakeholders monitor loan distribution trends and repayment efficiency over time.
